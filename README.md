
## Sobre o Projeto

Este repositório atua como um hub central de conhecimento e prática para o desenvolvimento em Python. O material aqui contido serve como base transversal para:

1.  **Apoio Acadêmico:** Suporte às disciplinas do curso de **Engenharia de Software da Universidade de Uberaba (Uniube)**, fornecendo exemplos práticos de algoritmos, estruturas de dados e paradigmas de programação.
2.  **Treinamento Corporativo:** Material de referência para capacitação profissional, com foco específico na criação de **scripts de automação** e otimização de processos operacionais.
3.  **Colaboração Interdisciplinar:** Espaço para integração entre aulas, projetos de extensão e workshops técnicos.

## Objetivos de Aprendizado

O conteúdo foi estruturado para guiar o aluno/profissional desde a lógica básica até a implementação de soluções de automação, visando:

* Solidificar o entendimento da sintaxe e das *buit-in functions* do Python.
* Aplicar boas práticas de desenvolvimento de software (Clean Code, PEP 8).
* Desenvolver scripts para automação de tarefas repetitivas (File System, Web Scraping, APIs).
* Introduzir conceitos de Engenharia de Software aplicados (Testes Unitários, Controle de Versão).

## Estrutura do Repositório

O conteúdo está organizado por módulos de complexidade crescente:

* `01_fundamentos/`: Sintaxe, variáveis e tipos de dados.
* `02_controle_fluxo/`: Estruturas condicionais e laços de repetição.
* `03_estruturas_dados/`: Listas, dicionários, tuplas e manipulação de strings.
* `04_programacao_modular/`: Funções, escopo e importação de módulos.
* `05_poo_avancada/`: Classes, herança, polimorfismo e encapsulamento.
* `06_automacao_scripts/`: Manipulação de arquivos (OS, Shutil), interação com S.O. e bibliotecas de automação.
* `07_integracoes/`: Consumo de APIs, conexão com bancos de dados e requests.

## Pré-requisitos e Ambiente

Para garantir a padronização entre o ambiente acadêmico e corporativo, recomenda-se:

* **Linguagem:** Python 3.8+
* **Controle de Versão:** Git (Configurado com credenciais institucionais ou pessoais).
* **IDE Recomendada:** VS Code ou PyCharm.
